Instruction building project with cmake :
Assuming user have downloaded all the sources code contained in a folder naming "m2mo_cpp"
Create beside it a folder named "build_m2mo_cpp". Go in to the folder "build_m2mo_cpp" and execute the command

prompt$> cmake -G"appropriated_generator" -D CMAKE_BUILD_TYPE=Debug ../m2mo_cpp

The appropriated generator for 
- Use of Eclipse under Linux = "Eclipse CDT4 - Unix Makefiles"
