build instructions (out-of-sources)

1.linux user must install ncurses-dev pakage

1.create a folder side by side with WINZAF, name it buildWINZAF
2.by terminal, go to the folder buildWINZAF and type the command

cmake ../WINZAF/ -G "Eclipse CDT4 - Unix Makefiles" -DCMAKE_BUILD_TYPE=Debug

3.open Eclipse CDT and import the project from the folder buildWINZAF, modify, compile

4.go to the build folder then execute ./test_winzaf
